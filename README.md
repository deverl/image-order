# Image order

A small local web app that lists images from the `img` directory and lets you reorder them with drag and drop. Order is randomized each time you load the page.

## Run locally

```bash
node server.js
```

Then open **http://localhost:9080**

## Usage

- Images in the `img` folder (jpg, jpeg, png, gif, webp) are shown in a grid.
- Drag a card and drop it on another to change the order.
- Order is randomized each time you load the page.
